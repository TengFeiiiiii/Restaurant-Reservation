# Restaurant-Reservation
基于微信的餐厅预约系统

大三上学期的课程设计。

基于ThinkPHP框架。
微信接口类使用
*  @author  dodge <dodgepudding@gmail.com>
 *  @link https://github.com/dodgepudding/wechat-php-sdk
 *  @version 1.2
 
 受限于大小，图片、数据库均未上传，如需进一步了解，可私戳。
 
 ## 特点
 - 登录、订餐、选座流程
 - 支持微信支付流程
 - 支持绑定微信公众号
 - 具有后台管理功能，配置餐厅、菜品
 - 适合课程设计
